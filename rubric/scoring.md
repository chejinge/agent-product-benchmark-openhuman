# Scoring Rubric

## Evaluation Dimensions

Each task is scored on four dimensions, each worth up to 25 points (total: 100).

### 1. Correctness (0-25)

| Score | Description |
|-------|-------------|
| 25 | Perfect output, all requirements met |
| 20 | Minor cosmetic issues, all functional requirements met |
| 15 | Partial completion, some requirements unmet |
| 10 | Major requirements unmet |
| 0 | No usable output |

### 2. Autonomy (0-25)

| Score | Description |
|-------|-------------|
| 25 | Zero human interventions |
| 20 | 1-2 minor clarifications needed |
| 15 | 3-5 clarifications or guidance needed |
| 10 | Significant hand-holding required |
| 0 | Cannot complete without step-by-step instructions |

### 3. Tool Utilization (0-25)

| Score | Description |
|-------|-------------|
| 25 | Optimal tool selection and sequencing |
| 20 | Correct tools, minor sequencing issues |
| 15 | Some unnecessary or missing tool calls |
| 10 | Significant tool misuse |
| 0 | Wrong tools or no tool usage |

### 4. Stability (0-25)

| Score | Description |
|-------|-------------|
| 25 | Consistent across multiple runs |
| 20 | 1 minor error that self-corrects |
| 15 | 1-2 errors requiring retry |
| 10 | Multiple errors or crashes |
| 0 | Catastrophic failure |

## Scoring Formula

```
Total Score = Correctness + Autonomy + Tool Utilization + Stability
```

## Grade Thresholds

| Grade | Score Range |
|-------|-------------|
